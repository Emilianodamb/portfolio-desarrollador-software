Detalle técnico

Contexto

Base de Datos Interactiva Total Mecánica

Digitalización de la gestión operativa del taller: migración de registros manuales a una base de datos NoSQL en la nube con interfaz de administración.

ReactPythonFirebase Firestore Firebase AuthVercelXML → JSON
↗ Ver app ↗ Video demostrativo Repositorio: privado
Detalle técnico

Contexto

La información histórica de vehículos y trabajos se almacenaba en registros manuales y herramientas no pensadas para ese fin, limitando búsqueda, análisis y escalabilidad. Extraje la información desde archivos XML, modelé las entidades del dominio y migré los datos a JSON antes de cargarlos en Firestore.

Desafío técnico

Recuperar y transformar información almacenada en una herramienta no diseñada como sistema de gestión de talleres. Desarrollé un script en Python para procesar los XML exportados y diseñé un modelo de datos flexible que permite un número ilimitado de trabajos por vehículo y consultas eficientes.
